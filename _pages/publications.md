## Selected Publications

- **VeloANN: SSD-Resident Graph ANN and VectorDB Storage-Execution Optimization** (arXiv'26) — SSD-resident graph ANN with storage-execution optimization for vector databases. [Paper](/publication/Arxiv-26-2)
- **TQEX(SQL): Tensor-based Query Engine Enhanced by Bridging the Gap** (SIGMOD'26) — Tensor-based SQL execution bridging irregular relational workloads and tensor runtimes. [Paper](/publication/SIGMOD-26-2)
- **GES: High-Performance Graph Processing Engine and Service in Huawei** (SIGMOD'25) — High-throughput graph processing engine and service with benchmark-leading LDBC SNB results. [Paper](/publication/SIGMOD-25-1)
- **TenGraph: A Tensor-Based Graph Query Engine** (VLDB'24) — Tensor-based graph query execution with GPU acceleration. [Paper](/publication/VLDB-24)
- **Parallel Query Processing: To Separate Communication from Computation** (SIGMOD'22) — Distributed SQL execution by explicitly separating communication and computation. [Paper](/publication/SIGMOD-22-1)
