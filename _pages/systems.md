## Research Systems

### SEMA (Semantic Query Engine)
A multimodal semantic data system for unified processing across structured and unstructured data, including semantic ETL and semantic ad-hoc querying.

- Focus: semantic query processing, multimodal data pipelines, AI-assisted analytics.
- Status: active.
- Page: [SEMA system page](/systems/sema/)

### TDB (Tensor-Centric DB)
A design direction for GPU/NPU-native data processing over tensor runtimes with support for SQL and graph-style workloads.

- Focus: tensor-centric execution, heterogeneous acceleration, structured + unstructured analytics.
- Status: active.
- Page: [TDB system page](/systems/tdb/)

### DISC (Distributed Subgraph Counting)
A distributed research system for general subgraph counting based on relational operators.

- Focus: distributed query execution and graph analytics.
- Publication: [VLDB 2020](/publication/VLDB-20)
